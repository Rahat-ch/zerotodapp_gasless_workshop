## Getting Started

Install Dependencies

```bash
npm install
# or
yarn 
# or
pnpm install
```

Create a .env file

```shell
cp .env-example .env;
```

Update with your Biconomy API key from our Dashboard!

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
